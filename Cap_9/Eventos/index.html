EVENTOS
que son los eventos? Cualquier cambio/accion de una pagina, desde que carga, aparecen las imagenes, se hace scroll, se mueve el mouse, damos click y demas ....
esto quiere decir que al asignarle codigo a un elemento que esta participando a un evento y asi hacer/cambiar/ejecutar algo
como aplicar/realizar un evento? SE PUEDE HACER DE 4 FORMAS

1. Directamente en el html
< button onclick="alert(`esto es un evento`)" > Apretame

2. LLamando a una funcion en el html para que en el archivo js se ejecute
< button onclick="funcionClick" > Apretame

3. En el html se crea la clase para que en js se agregue el evento
HTML
< button class="boton" > Apretame
JavaScript
boton1= document.querySelector(".boton")
boton1.addEventListener("click",saludo);
function saludo (){alert("mensaje");}

4. En el html se crea la clase para que en js se agregue el evento
HTML
< button class="boton" > Apretame
JavaScript
boton1= document.querySelector(".boton")
boton1.addEventListener("click",()=>{
alert("mensaje");}
